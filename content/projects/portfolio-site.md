## О проекте

Одностраничный сайт-портфолио с акцентом на чистый дизайн и плавные анимации. Создан для демонстрации профессионального опыта и пет-проектов.

## Особенности

- **Минималистичный дизайн** — тёмная тема, tech-стиль, много воздуха
- **Плавные анимации** — scroll-эффекты, fade-in, hover-состояния через Framer Motion
- **Адаптивность** — mobile-first подход, работает на всех устройствах
- **Таймлайн опыта** — визуальное представление карьерного пути
- **Markdown для проектов** — удобное добавление новых кейсов

## Стек

Next.js 16, TypeScript, Tailwind CSS v4, shadcn/ui, Framer Motion
